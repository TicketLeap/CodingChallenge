Vehicle locations

The SEPTA Tracker 9000 is really cool. But right now it doesn't do a whole lot. How you do it is up to you, but here are the "must haves":

  • A marker for each SEPTA vehicle
  • An InfoWindow (see this) for each vehicle that includes:
    • Position (latitude and longitude)
    • (if defined) Vehicle ID
    • (if defined) Direction
    • (if defined) Destination
    How you present this information is totally up to you (see the markup for example on how to use InfoWindow's)
Bonus Round! If you're a JavaScript/Django wizard then try adding one or more these features:
  • Auto-refresh the map periodically with latest vehicle positions.
  • Allow the user to enter a position (lat+lng) along with a radius, and then only show vehicles within that area. How users input the data is your decision.
  • Allow the user to enter a position (lat+lng), and then report the closest vehicle to it.
